## How to Add Yourself

We use [All Contributors](https://allcontributors.org/) to recognize all contributions:

1. Add your contribution using the emoji key in a PR comment:
   ```
   @all-contributors please add @username for code
   ```
2. Or comment with: `@all-contributors please add @username for doc`
